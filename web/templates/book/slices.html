Срезы

Slice - это обёртка над массивом, которая хранит указатель на массив, длину и ёмкость.

Отличие nil-среза от пустого среза

  • nil slice - это срез, который не указывает ни на какой массив.
  • Пустой slice - это срез, указывающий на реальный массив длиной 0.
Свойство nil slice Пустой slice
Инициализация var s []int s := []int{}
Состояние s == nil это true s == nil это false
Длина len(s) == 0 len(s) == 0
Емкость cap(s) == 0 cap(s) == 0
Использование Можно использовать с `append` без проблем Можно использовать с `append` без проблем
