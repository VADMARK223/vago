slice - срез []int{1, 2, 3}
map - карта map[string]int
string - строка immutable, из изменять нельзя
struct - структура

Другие

Изменения в самом языке

Новый семантика переменных в цикле for

  • Теперь переменные, объявленные в цикле for, создаются заново на каждой итерации. Это устраняет распространённую «подводную камень» с замыканиями в циклах (например, при запуске горутин). Раньше одна переменная обновлялась каждый раз, что приводило к неожиданным результатам в замыканиях — теперь такая ошибка исключена по умолчанию.

for … range over integers

  • Добавлена возможность писать циклы вроде:
for i := range 10 {
    fmt.Println(i)
}
